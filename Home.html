Project Idea

Our project is based on our senior project and aims to develop an intelligent web-based tool that predicts the type of cancer using gene expression data. The tool will focus on the most common cancer types in Saudi Arabia, including breast, colorectal, thyroid, liver, and kidney cancers. The tool will assist oncologists in diagnosing cancer by analyzing gene expression profiles in CSV files.

Website functions

Here we present the functions which will be included in the wesite.

Users will be able to:
﻿- Create an account.
﻿- Long in and out of the account.
﻿- Edit the profile.
﻿- Upload gene expression file for analysis.
﻿- View analysis results.
﻿- Keep history of analyzed files and their results.
